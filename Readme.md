# Simple Heart

A pure C recreation of a heart-beating animation program.

[Python Version](python/heart_gaotian.py) reproduced based on [Tian Gao](https://github.com/gaogaotiantian) 's [video](https://www.bilibili.com/video/BV16g411B7Ff).

## Usage

```powershell
gcc heart.c -o heart.exe -lgdi32
```

Enjoy it!
